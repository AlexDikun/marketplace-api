# Техническое задание

## Общая информация

Веб-приложение с клиентским интерфейсом (Swagger / Postman) и серверной частью (backend).  
Взаимодействие осуществляется через REST API.

---

## Цель проекта

Разработка электронной доски объявлений для демонстрации практических навыков разработки, полученных в рамках курса  
**Java Back-end Developer (SolarLab)**.

---

## Технологический стек

- Java 21  
- Spring Framework (Spring Boot, Spring MVC, Spring Data JPA)  
- PostgreSQL  
- Keyloack
- Docker  

---

## Функциональные требования

### Пользователь (неавторизованный)
 
- может авторизоваться в системе
- может просматривать все объявления
- может просматривать все изображения под объявлением
- может просматривать все комментарии под объявлением   
- может искать объявления по ключевому слову

---

### Авторизованный пользователь

- может создавать объявления  
- может редактировать свои объявления  
- может удалять свои объявления
- может просматривать контактные данные других пользователей  
- может оставлять комментарии к объявлениям  

---

### Администратор

- может удалять любые объявления
